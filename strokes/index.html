同一笔，三种叫法

笔画是文字最小的零件——一笔的运动。汉字和日本汉字共用同一套笔画，但每种语言叫法不同。 Strokes are the smallest parts of the script — a single pen movement. Chinese hanzi and Japanese kanji share the same strokes; only the names differ. The red dot traces each stroke's direction and order.

Wanikani's stroke names are all shape-mnemonics (一 = Ground, 亅 = Barb) — handy for memorising but not real cross-language terms. Tap ▶ to hear the example character in each language. Wanikani 给笔画起的名字都是形状助记（一 = Ground，亅 = Barb）， 记忆方便但不通用。点 ▶ 听例字发音。
